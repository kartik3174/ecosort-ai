# **App Name**: EcoSort AI

## Core Features:

- Litter Reporting: Allows citizens to submit litter reports with photo capture, location detection, and waste tagging.
- Hazardous Waste Detection: AI tool for real-time identification of hazardous waste materials from user-submitted images, ensuring immediate alert prioritization.
- Interactive Litter Map: Displays a real-time map of reported litter locations with color-coded pins for different waste statuses (Still There, Cleaned, Hazardous, Recyclable).
- Role-Based Access Control: Implements distinct dashboards and functionalities for citizens and city officials, ensuring secure and streamlined operations.
- Citizen Dashboard: Presents personalized statistics and actions for citizen users, including reports submitted, locations cleaned, and eco contribution score.
- City Official Dashboard: Provides city officials with comprehensive tools for managing litter reports, assigning cleanup teams, and monitoring hazardous alerts.
- Analytics and Reporting: Offers analytical insights through charts on litter reports by area, cleanup performance, hazard frequency, and monthly trends.

## Style Guidelines:

- Primary color: Forest green (#38A3A5) to evoke nature and environmental awareness.
- Background color: Light pastel green (#E0EFE6) to create a clean and eco-friendly atmosphere.
- Accent color: Sky blue (#74B4E0) to represent cleanliness and highlight interactive elements.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif.
- Use simple, eco-friendly icons for litter types and dashboard actions.
- Implement a modern dashboard layout with rounded cards, soft shadows, and glassmorphism effects for a professional and visually appealing interface.
- Incorporate subtle hover animations on buttons and interactive elements to enhance user engagement.
